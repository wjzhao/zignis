
# Zignis SQL审核平台


## API集成
- **Inception**: 去哪儿网开源，提供SQL语句审核、执行、回滚功能
- **SQLAdvisor**: 美团开源，提供分析SQL中的where条件、聚合条件、多表Join关系，输出索引优化建议
- **SOAR**: 小米开源，提供SQL启发式算法的语句优化、多列索引优化等功能

## 功能简介

- 目标库管理
    - 支持多场地/数据中心的数据库管理，集群方式归纳目标数据库
    - 支持目标数据库配置，库/表结构查询
- SQL操作

- SQL查询
